# Simple Calculator Project

This is a simple Python project that works as a basic calculator. It takes two numbers and an operation from the user and displays the result.

## Project Description

This program asks the user to enter:

* First number
* Operation (+, -, *, /)
* Second number

After that, the program performs the selected operation and prints the result.

If the user tries to divide by zero, the program displays a message instead of showing an error.

## Features

* Takes input from the user
* Supports basic operations:
  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Checks division by zero
* Displays the result instantly

## Language Used

This project is made using:

* Python
